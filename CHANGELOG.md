## [1.0.0] - 2024-02-17

### 新增
- ✨ 首次发布
- 🎙️ 语音录制功能，支持 macOS/Windows/Linux
- 🤖 集成 Groq Whisper API 进行语音识别
- ✨ 集成 DeepSeek API 进行 AI 文本处理
- ⌨️ 自动键盘输入，支持剪贴板降级
- 🎨 悬浮球 UI，支持拖动和状态显示
- ⚙️ 设置窗口，支持完整的配置管理
- 📚 历史记录功能，支持搜索和导出
- ⚡ 全局快捷键系统，支持自定义
- 🔔 系统托盘集成
- 🌐 中英混说识别支持
- 📝 多种 AI 处理模式（清理/格式化/翻译/助手）
- 🎭 个性化语调风格（正式/随意/专业/创意）
- 📖 个人词典功能
- 🔐 本地加密存储
- 📖 完整的文档（开发指南、API指南、用户指南）

### 技术特性
- Electron + TypeScript + React 技术栈
- 模块化架构设计
- 完整的类型定义
- IPC 通信封装
- 错误处理机制
- 跨平台支持

## 版本说明

### 版本号格式
`主版本号.次版本号.修订号`

- **主版本号**：重大更新，可能不兼容
- **次版本号**：新功能，向下兼容
- **修订号**：问题修复，向下兼容

### 标签说明
- ✨ 新功能
