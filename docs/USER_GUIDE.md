# Typeless 用户指南

## 简介

Typeless 是一款 AI 语音输入工具，可以将你的口语化语言自动整理成条理清晰、有层次的文字。无论是写文章、做笔记还是编程，Typeless 都能帮助你提高输入效率。

## 核心功能

- 🎙️ **语音输入** - 说话比打字快 5 倍
- 🤖 **智能处理** - 自动去除口语化表达
- 🌐 **中英混说** - 准确识别中英文混合语音
- ⌨️ **自动输入** - 文本自动插入光标位置
- 📚 **历史记录** - 保存所有处理记录

## 快速开始

### 1. 安装

下载对应平台的安装包并安装：

- **macOS**: `Typeless-x.x.x.dmg`
- **Windows**: `Typeless-x.x.x.exe`
- **Linux**: `Typeless-x.x.x.AppImage`

### 2. 配置 API Keys

首次使用时需要配置 API Keys：

1. 右键点击悬浮球
2. 选择"设置"
3. 在"API 配置"标签页中：
   - 输入 Groq API Key
   - 输入 DeepSeek API Key
4. 点击"保存"

> 💡 不知道如何获取 API Key？查看 [API 使用指南](./API.md)

### 3. 开始使用

**方式一：使用快捷键**
- 默认快捷键：`Command/Ctrl + Shift + R`
- 按下开始录音，再次按下停止录音

**方式二：点击悬浮球**
- 点击悬浮球开始录音
- 再次点击停止录音

**方式三：使用系统托盘**
- 点击系统托盘图标
- 选择"开始/停止录音"

## 界面说明

### 悬浮球

悬浮球会显示当前状态：

- **灰色** - 空闲状态，可以点击开始录音
- **红色** - 正在录音，显示录音时长
- **蓝色** - 正在处理，请稍候
- **黄色** - 出现错误，查看提示信息

### 设置窗口

设置窗口包含以下标签页：

#### API 配置
- 配置 Groq API Key
- 配置 DeepSeek API Key
- 测试 API 连接

#### 快捷键
- 自定义全局快捷键
- 支持多种组合键

#### 个性化
- 选择语调风格（正式/随意/专业/创意）
- 设置默认语言
- 管理个人词典

#### 其他
- 开机自启
- 悬浮球透明度
- 历史记录保留天数

### 历史记录窗口

- 查看所有处理记录
- 搜索历史记录
- 复制文本内容
- 导出为 Markdown 或文本文件

## 使用技巧

### 1. 提高识别准确率

- **清晰发音** - 保持适当的语速，避免含糊
- **减少噪音** - 在安静的环境中使用
- **使用个人词典** - 添加专业术语和专有名词

### 2. 优化处理效果

- **选择合适的语调风格**
  - 正式：适合商务、学术场景
  - 随意：适合日常交流
  - 专业：适合技术文档
  - 创意：适合写作创作

- **控制录音时长**
  - 建议每次 10-60 秒
  - 过短可能导致识别不准确
  - 过长可能影响处理效果

### 3. 快捷键使用

- **开始/停止录音** - `Cmd/Ctrl + Shift + R`
- **快速翻译** - `Cmd/Ctrl + Shift + T`
- **AI 助手** - `Cmd/Ctrl + Shift + A`

### 4. 个人词典

添加常用词汇可以提高识别准确率：

1. 打开设置窗口
2. 进入"个性化"标签页
3. 在"个人词典"中添加词条

建议添加：
- 专业术语
- 人名地名
- 产品名称
- 公司名

## 常见问题

### 录音没有反应

**可能原因**：
- 麦克风权限未开启
- 麦克风设备未连接
- 其他应用占用了麦克风

**解决方案**：
1. 检查系统麦克风权限
2. 确认麦克风设备正常
3. 关闭其他使用麦克风的应用

### 识别结果不准确

**可能原因**：
- 发音不清晰
- 背景噪音太大
- 使用了专业术语

**解决方案**：
1. 在安静的环境中使用
2. 放慢语速，清晰发音
3. 将专业术语添加到个人词典

### 文本没有输入到目标应用

**可能原因**：
- 目标应用不支持输入
- 权限不足
- 焦点不在目标应用

**解决方案**：
1. 确保目标应用处于焦点状态
2. 检查辅助功能权限（macOS）
3. 文本已复制到剪贴板，可以手动粘贴

### API 错误

**可能原因**：
- API Key 无效
- 网络连接问题
- API 配额不足

**解决方案**：
1. 检查 API Key 是否正确
2. 检查网络连接
3. 查看 API 使用配额

## 隐私与安全

- 🔒 **本地存储** - 所有设置和历史记录存储在本地
- 🔐 **加密传输** - 所有 API 通信使用 HTTPS 加密
- 🚫 **不收集数据** - 不会收集用户数据用于训练
- 🗑️ **自动清理** - 历史记录自动清理过期数据
